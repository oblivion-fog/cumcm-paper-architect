# 题型差异化详细指南

## A 题（机理建模/工程物理）

**特点：** 偏微分方程、有限元、热传导、力学分析等物理机制为核心。
**论文重心：** 物理背景 → 机理推导 → 数值方法 → 仿真验证。

```
额外要求：
- 物理背景小节（热传导定律、力学原理的文献引用）
- 偏微分方程的定解条件（初始条件+边界条件）详细推导
- 数值求解方法的选择理由（为什么用有限差分/有限元？）
- 网格无关性/步长收敛性验证
- **模型准备节（2025 A196 模式）：** 在模型建立前设独立节建立坐标系、运动轨迹方程、几何关系等通用基础，供所有子问题复用
```

参考论文风格：
- 2018 A 题(高温作业服装) — 从 Fourier 定律出发逐层推导热传导方程，物理背景占一节篇幅。
- 2015 A 题(太阳影子定位) — 从太阳高度角、赤纬角、时角等天文学参数推导影长变化模型。
- **2025 A 题(无人机烟幕遮蔽) — 设"五、模型准备"节建立空间直角坐标系+运动轨迹方程，后续5个子问题全部基于此框架。**

## B 题（优化调度/资源配置）

**特点：** 整数规划、组合优化、调度算法、博弈论。
**论文重心：** 决策变量定义 → 目标函数构造 → 约束条件完整列举 → 求解器/算法选择 → 约束违反检查。

```
额外要求：
- 每个约束条件必须解释实际含义（不是只写公式）
- 求解器参数（MIPGap、时间限制、warm start）说明
- 约束可行性验证结果表（每类约束一行，违反数=0）
- 若用启发式算法，需配收敛图+最优性证明或对比
```

参考论文风格：2017 B 题(拍照赚钱任务定价) — 基于最大流的启发式算法，含定价规律建模和任务分配优化模型。2023 B 题(多波束测深) — 贪心算法+仿真验证。

## C 题（数据分析/统计建模）

**特点：** 回归分析、分类、聚类、时间序列、机器学习。
**论文重心：** 数据探索 → 特征工程 → 模型选择 → 评估指标 → 对比实验。

```
额外要求：
- 数据分布可视化（直方图/箱线图/相关性热力图）
- 数据清洗逻辑详细说明（异常值标准和处理方式）
- 评估指标选择理由（为什么用WAPE不用RMSE？）
- 多模型对比表（至少2个baseline模型）
- 过拟合/欠拟合讨论
```

参考论文风格：2018 C 题(百货商场会员画像) — RFMS 指标构建+K-means 聚类+价值分类模型。

## D/E 题（综合评价/决策分析）

**特点：** 层次分析法、TOPSIS、模糊评价、决策树。
**论文重心：** 指标体系构建 → 权重确定（主观+客观）→ 评价模型 → 灵敏度验证。

```
额外要求：
- 指标体系图或表
- 权重方法的合理性论证
- 评价结果对权重变化的灵敏度
```

## 实战做法

第一眼看到赛题即判断题型，对照上述要求确认论文结构，在问题分析中体现题型特征。

对于递进结构的题目（如 Q1→Q2→Q3），确保：
- 问题分析中明确说明依赖关系
- Q2 模型建立前引用 Q1 结果
- Q3 模型建立在 Q2 最优解基础上
